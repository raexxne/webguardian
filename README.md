## 📌 Overview

**Web Guardian** is a Dynamic Application Security Testing (**DAST**) web application vulnerability scanner developed to help users identify and understand security weaknesses in web applications.

It integrates powerful security tools such as **OWASP ZAP**, combined with a custom-built Flask web application, to deliver an efficient and user-friendly vulnerability assessment system.

---

## 🚀 Key Features

- 🔍 **Manual Target URL Scanning**  
  Users can manually enter a target URL and initiate the scan to detect common web vulnerabilities.

- 📊 **Comprehensive Reporting**  
  View organized scan results categorized by severity levels (High, Medium, Low).

- 🔔 **Notification System**  
  Receive alerts and updates for scan activities and system events.

- 👥 **Role-Based Access Control (RBAC)**  
  Manage system access for Admin, Manager, and Staff roles.

---

## 🔐 Security Scope

Web Guardian focuses on detecting common web vulnerabilities such as:

- SQL Injection (SQLi)
- Cross-Site Scripting (XSS)
- Security Misconfigurations
- Sensitive Data Exposure
